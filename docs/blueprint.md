# **App Name**: CEPLA Digital

## Core Features:

- Static Page Generation: Create and host static pages providing information about Capilla San Jos√©, CEPLA.
- News Carousel: Implement a news carousel on the homepage to display the latest news and announcements.
- Ministry Login: Secure login functionality for ministry coordinators (music, liturgy, altar servers, etc.) to manage their respective areas.
- Ministry Application Form: Implement a form for users to apply and join a specific ministry within the chapel.
- Resource Upload and Management: Allow authorized users to upload and manage resources such as missals, prayers, and hymns.
- Content Generation Tool: AI-powered tool that helps ministry coordinators generate content for their sections by providing inspiration from similar Catholic texts. The AI tool will use reasoning to include snippets from those texts based on relevance.

## Style Guidelines:

- Primary color: Deep Gold (#D4AF37) to reflect the traditional and sacred nature of the chapel. 
- Background color: Off-White (#F5F5DC), to give a feeling of old parchment and create a serene, traditional aesthetic. 
- Accent color: Muted Red (#A52A2A) used sparingly for calls to action and important links, complementing the primary color. 
- Headline font: 'Literata' serif, to provide an intellectual and slightly vintage feel. 
- Body font: 'Inter' sans-serif, for clear and modern readability, complementing the serif headline font. 
- Use classic, elegant icons inspired by Gregorian chant aesthetics to represent different sections and actions. 
- Use a clean, structured layout with a focus on readability and ease of navigation. 
- Subtle fade-in animations for content loading and transitions.
